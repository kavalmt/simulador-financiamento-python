
# Simulador de Financiamento em Python 
Este projeto simula um financiamento bancário permitindo calcular:
- Valor da parcela mensal
- Montante total pago
- Tabela de amortização detalhada
---
# Conceitos aplicados:
- Juros compostos
- Sistema de amortização
- Cálculo de parcelas (modelo utilizado por bancos)
---
## Exemplo de saída:
Parcela 1 | Juros: R$ 50.00 | Amortização: R$ 79.50 | Saldo: R$ 920.50  
Parcela 2 | Juros: R$ 46.02 | Amortização: R$ 83.48 | Saldo: R$ 837.02  
---
